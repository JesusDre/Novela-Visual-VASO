SUGERENCIA PRACTICA PARA SIMULAR AUTORIAS EN GIT
==================================================

Antes de cada commit, cambia autor localmente:
- git config user.name "Chucho"
- git config user.email "chucho@equipo.local"

Alternativa en una sola linea por commit:
- git -c user.name="Chucho" -c user.email="chucho@equipo.local" commit -m "feat(db): ..."

Tip para evitar errores
- Haz commit solo de los archivos asignados en cada paso (git add <archivo1> <archivo2> ...).
- Sigue la secuencia cronologica para que el historial se vea natural.
